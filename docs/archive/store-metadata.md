**APP NAME:** Orbital Traffic

**SUBTITLE (30 chars max):** Live Satellite & Space Tracker

**DESCRIPTION:**

Track every satellite, space station, and asteroid near Earth on an interactive 3D globe.

Orbital Traffic displays 18,000+ tracked objects — from the International Space Station to Starlink constellations to hazardous near-Earth asteroids — with orbital element sets refreshed every 20 minutes from CelesTrak and Space-Track.org. Positions are computed on your device using industry-standard SGP4 orbital propagation.

**Features:**

- 3D interactive Earth globe — rotate, zoom, and explore every orbit

- 18,000+ objects across 8 classes: Stations, Navigation, Geostationary, Starlink, Science, Rocket Bodies, Debris, and Hazardous NEOs

- Object detail panels with altitude, speed, orbit regime, launch data, and operator

- ISS special panel with current crew, expedition info, and mission updates

- Time Machine — replay orbital history or simulate future positions at 10×, 60×, or 300× speed

- Satellite pass predictions — see when any object passes over your location

- Search by object name or NORAD catalog ID

- Color-coded, shape-distinguished object classes for accessibility

**Data accuracy note:** Orbital positions are derived from Two-Line Element sets (TLEs) typically updated every 20–60 minutes from CelesTrak. Propagated positions are estimates based on the most recent available element sets. Orbital Traffic is intended for educational and informational use and should not be used for operational space safety decisions.
